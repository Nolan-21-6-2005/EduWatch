### Cài đặt github
Gói cài đặt: https://git-scm.com/install/windows

Hoặc sử dụng winget
```powershell
winget install --id Git.Git -e --source winget
```

Clone:
```
git clone https://github.com/Nolan-21-6-2005/BehaviorNet.git
```
